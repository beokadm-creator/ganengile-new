# 가격 비교 분석: 역삼 GS타워 → 정자 두산위브파빌리온

## 경로 정보

**출발지**: 역삼 GS타워 (역삼역 2호선)
**도착지**: 정자 두산위브파빌리온 (정자역 분당선)

### 실제 경로
- 역삼역 (2호선) → 강남역 → 선릉역 → 수서역 (3호선 환승)
- 수서역 → 가락시장역 → 경기광주역 → 정자역 (분당선 환승)
- **총 역 개수**: 7개역
- **지하철 거리**: 약 18.5km
- **도로 거리**: 약 20.2km
- **예상 소요시간**: 45-60분
