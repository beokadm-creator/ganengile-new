# “내용 채운” 보드 SVG 4종 (Figma Import용)

말씀하신 대로 **도형만 있는 템플릿이 아니라, 컬럼/카드/액션/예외까지 예시로 채운 버전**을 만들어뒀습니다.  
Figma에서 **붙여넣기**가 아니라 **Import 또는 드래그&드롭**으로 넣어야 내용이 그대로 들어갑니다.

## 파일
1) `ganengile_e2e_swimlane_filled.svg`
2) `ganengile_customer_board_filled.svg`
3) `ganengile_giller_board_filled.svg`
4) `ganengile_ops_board_filled.svg`

## Figma에 넣는 방법(중요)
1. Figma에서 `Mission Board` 페이지(없으면 새로 생성)로 이동
2. **File → Import…** 또는 파일을 캔버스에 **드래그&드롭**
3. 들어간 뒤, 전체 선택 후 **Shift+2(선택영역로 줌)**  
4. 필요하면 전체를 Frame으로 감싸고 이름만 맞추면 됩니다:
   - `01_Overview / E2E Swimlane`
   - `02_Boards / Customer Board`
   - `02_Boards / Giller Board`
   - `02_Boards / Ops Board`
